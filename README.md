# Raspberry Pi Button Controller

Simple Python project to read a button connected to a Raspberry Pi GPIO pin.

## Hardware

Button connected to:

GPIO10 (Physical pin 19)

Use internal pull-up resistor.

Wiring:

GPIO10 ---- Button ---- GND
